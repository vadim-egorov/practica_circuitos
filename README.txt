Esto es un repositorio con el programa para realizar
la practica 4 de la asignatura Circuitos electricos.

El fichero "practica4.py" contiene el codigo en Python
para resolver Tarea 1, en particular:
------------------------------------------------------------
Calcula la frecuencia de resonancia.

Hace la tabla 4.2 calculando los valores teoricos 
y mostrando los valores experimentales del fichero de texto.

Dibuja la diagrama de Bode de la funcion de transferencia
(V_s/V_g) de frecuencia en la escala logaritmica.
------------------------------------------------------------

El programa se puede abrir utilizando Spyder 
o cualquier otro IDE y interpretador de Python
teniendo numpy y matplotlib instalados.
------------------------------------------------------------

Para ajustar el programa tienes que sustituir los valores
de las variables "inductancia", "capasitancia", "resistencia_g"
por los datos medidos en el laboratorio
------------------------------------------------------------

Tienes que sustituir los datos experimentales
en el fichero "Table 4.2 Experimental.txt".

El programa lee los datos a partir de cuarta linea,
entonces no hace falta borrar primeras tres lineas.

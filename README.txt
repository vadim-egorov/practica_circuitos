Esto es un repositorio con el programa para realizar
la practica 4 de la asignatura Circuitos electricos.

Calcula la frecuencia de resonancia.

Hace la tabla 4.2 calculando los valores teóricos y mostrando los valores experimentales del fichero de texto.
Dibuja el diagrama de Bode de la función de transferencia
(Vs/Vg)(Vs​/Vg​)
en escala logarítmica.
El programa se puede abrir utilizando Spyder o cualquier otro IDE e interpretador de Python, teniendo numpy y matplotlib instalados.
Para ajustar el programa tienes que sustituir los valores de las variables "inductancia", "capacitancia", "resistencia_g" por los datos medidos en el laboratorio.

Tienes que sustituir los datos experimentales en el fichero "Table 4.2 Experimental.txt".

El programa lee los datos a partir de la cuarta línea, así que no hace falta borrar las tres primeras líneas.
